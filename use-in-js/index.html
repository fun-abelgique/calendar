Paste the above code into your HTML, before the rest of your <script> tags, and you can use Calendar! in your JavaScript.

Type in the following code: const myDateFactorial = new DateFactorial(); and you'll be good to go!

You can use DateFactorial() like a Date() object. You can get the year, year in the four year cycle, day of the year (you can do this for the Gregorian calendar too!), month, day of the month, day of the week, and a nice date string, like the one above. You can also use DateFactorial() as a replacement for the Date() object. You can use new DateFactorial().gDate instead!

Below is a table of what you can do with DateFactorial():

Method or Property Explanation
getYear() Returns the current year.
getWhichYear() Returns what year it currently is in the four year cycle as a number from 0 to 3.
getBonusDay() Returns "Bonus" if it's a bonus day, "Bonus Bonus" if it's a bonus bonus day, or an empty string if it isn't a bonus day.
getDayOfYear() Returns the day of the year as a number from 1 to 364, but returns the return value of getBonusDay() if it's a bonus day.
getGregorianDayOfYear() Returns the day of the year in the Gregorian calendar as a number from 1 to 365.
getDate() Returns the day of the month as a number from 1 to 28.
getMonth() Returns the month as a number from 0 to 12.
getDay() Returns the day of the week as a number from 0 to 6.
toDateString() Returns a readable date string containing the day of the week, month, day of the month, year in the four year cycle, year, and either C! or BC!.
gDate A standard Date() object.
leapYear A Boolean that will be true when it is a leap year in the Gregorian calendar.
